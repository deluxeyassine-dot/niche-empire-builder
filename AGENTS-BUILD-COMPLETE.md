## Usage Examples

### Basic Usage

```typescript
import { TrendScoutAgent } from './agents';

const agent = new TrendScoutAgent(
  process.env.ANTHROPIC_API_KEY!,
  process.env.SUPABASE_URL!,
  process.env.SUPABASE_KEY!
);

// Run complete workflow
const results = await agent.runComplete();
```

### Full Automation Workflow

```typescript
import * as Agents from './agents';

// 1. Discover trending niche
const trendScout = new Agents.TrendScoutAgent(apiKey, url, key);
const trends = await trendScout.scanTrendingTopics(platforms);

// 2. Validate opportunity
const validator = new Agents.OpportunityValidatorAgent(apiKey, url, key);
const validation = await validator.validateNiche(trends[0].topic);

// 3. Create product
const courseBuilder = new Agents.CourseBuilderAgent(apiKey, url, key);
const course = await courseBuilder.runComplete(trends[0].topic);

// 4. Generate content
const videoCreator = new Agents.VideoCreatorAgent(apiKey, url, key);
const video = await videoCreator.runComplete({ topic: trends[0].topic });

// 5. Publish everywhere
const publisher = new Agents.MultiPlatformPublisherAgent(apiKey, url, key);
await publisher.publishToAll(video);

// 6. Track performance
const analytics = new Agents.PerformanceAnalyticsAgent(apiKey, url, key);
const metrics = await analytics.trackMetrics();
```
